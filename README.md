# Dacon_AI_Competition  

*데이콘 AI 경진대회 참여한 경험을 기록합니다.*  🤓


  
  
- BiLSTM을 이용한 영화 리뷰 감성 분류(~20220121)

  - train set과 val set을 8:2로 구성하여 학습.
  - train acc 98.72%, val acc 84.65% 을 기록.
